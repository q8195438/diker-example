# diker-example
The project shows how to use diker

## Dependencies
- [Node.js](https://nodejs.org)
- [Docker](https://www.docker.com)

## How to use
```bash
# Terminal-0
git clone https://github.com/Eronana/diker-example.git
cd diker-example
npm i

# Terminal-1
npm run init

# Terminal-2
npm run watch

# Terminal-3
npm run dev
```
